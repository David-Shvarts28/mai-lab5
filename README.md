# Лабораторная работа №4: Симуляция библиотеки

Система управления библиотекой с пользовательскими коллекциями и псевдослучайной симуляцией.

## Описание

Проект реализует систему управления библиотекой с двумя пользовательскими коллекциями:
- **BookCollection** - списковая коллекция для хранения книг и журналов
- **IndexDict** - словарная коллекция для индексации по ISBN, автору и году

## Запуск

```bash
python3 -m src.main
```

С параметрами:

```bash
python3 -m src.main 20

python3 -m src.main 20 123
```

## Тестирование

```bash
python3 -m unittest discover tests
```
